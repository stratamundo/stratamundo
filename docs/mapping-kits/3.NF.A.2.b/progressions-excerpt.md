# Progressions excerpt — 3.NF.A.2.b

**Source:** Progressions-3-5-Fractions.pdf, §Grade 3 — "The number line and number line diagrams," extracted pages 8–9.

## Verbatim excerpt

> To construct a unit fraction on a number line diagram, e.g., 1/3, students partition the unit interval into 3 intervals of equal length and recognize that each has length 1/3. They determine the location of the number 1/3 by marking off this length from 0, and locate other fractions with denominator 3 by marking off the number of lengths indicated by the numerator. 3.NF.2 (p. 9)

> The number line reinforces the analogy between fractions and whole numbers. Just as 5 is the point on the number line reached by marking off 5 times the length of the unit interval from 0, so 5/3 is the point obtained in the same way using a different interval as the unit of measurement, namely the interval from 0 to 1/3. (p. 9)

Sidebar — the standard verbatim:

> 3.NF.2 Understand a fraction as a number on the number line; represent fractions on a number line diagram.
>
> b. Represent a fraction a/b on a number line diagram by marking off a lengths 1/b from 0. Recognize that the resulting interval has size a/b and that its endpoint locates the number a/b on the number line. (p. 9)

The Progression's diagram (p. 9) shows a number line marked off in thirds from 0/3 through 12/3, illustrating that fractions a/b extend past 1 along the number line.

## Extraction for drafters

**What the kid should be able to do**

- Locate a/b on the number line by marking off a copies of length 1/b starting at 0.
- Recognize that the interval from 0 to a/b has size a/b and the right endpoint of that interval IS the point a/b.
- Plot fractions both less than and greater than 1 (e.g., 5/3 lands between 1 and 2).

**Common misconceptions / developmental hurdles**

- Not explicitly named in this section. The Progressions emphasizes (per 3.NF.A.1, p. 8) that "there is no need to introduce 'proper fractions' and 'improper fractions' initially" — 5/3 is just 5 copies of 1/3 marked off from 0. Treating fractions greater than 1 as a separate, harder case is a teaching trap. Consult `fractions-misconceptions.json` for SM's project-internal list.

**Where this skill sits in the progression**

- Earlier: 3.NF.A.2.a (locate 1/b on the number line), 3.NF.A.1 (a/b as a copies of 1/b).
- Later: 3.NF.A.3.a (equivalent fractions as the same point on the number line), 4.NF.A.1 (general equivalence rule), 4.NF.A.2 (compare fractions with unlike denominators).

**Recommended representations / models named in the Progressions**

- Number line diagrams partitioned into b equal subintervals across each whole-number interval (e.g., thirds running 0/3, 1/3, 2/3, 3/3, 4/3, ..., 12/3).
- "Marking off a lengths of 1/b from 0" as the construction action.
- Tape diagrams and area representations as parallel/bridging models.
