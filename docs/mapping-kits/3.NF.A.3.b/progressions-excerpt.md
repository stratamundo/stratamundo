# Progressions excerpt — 3.NF.A.3.b

**Source:** Progressions-3-5-Fractions.pdf, §Grade 3 — "Equivalent fractions," extracted pages 9–10.

## Verbatim excerpt

> Equivalent fractions. Grade 3 students do some preliminary reasoning about equivalent fractions, in preparation for work in Grade 4. As students experiment on number line diagrams they discover that many fractions label the same point on the number line, and are therefore equal; that is, they are equivalent fractions. For example, the fraction 1/2 is equal to 2/4 and also to 3/6. Students can also use tape diagrams to see fraction equivalence. 3.NF.3 (p. 9)

Sidebar — the standard verbatim:

> 3.NF.3 Explain equivalence of fractions in special cases, and compare fractions by reasoning about their size.
>
> b. Recognize and generate simple equivalent fractions, e.g., 1/2 = 2/4, 4/6 = 2/3. Explain why the fractions are equivalent, e.g., by using a visual fraction model. (p. 9)

Adjacent diagrams on pp. 9–10 show 1/4, 2/4, 3/4, 4/4 = 1 on one number line and 1/2, 2/2 = 1 on a second number line; and 1/6 stacked under 1/2, supporting reasoning that 3/6 = 1/2.

## Extraction for drafters

**What the kid should be able to do**

- Recognize simple equivalent fractions (e.g., 1/2 = 2/4, 4/6 = 2/3).
- Generate at least one equivalent fraction for a given simple fraction using a visual model (number line, tape, area).
- Explain WHY the two fractions are equivalent with a visual fraction model, not just by rule.

**Common misconceptions / developmental hurdles**

- Not explicitly named in this section. The Progressions emphasizes that Grade 3 work here is **preliminary** ("preparation for work in Grade 4") — the formal rule a/b = (n×a)/(n×b) is a Grade 4 expectation, NOT Grade 3 (p. 9, p. 11). Drafters should not push 3rd graders toward the general multiplication rule; reasoning is grounded in visual models. Consult `fractions-misconceptions.json` for SM's project-internal list.

**Where this skill sits in the progression**

- Earlier: 3.NF.A.3.a (equivalence = same point on number line / same size), 3.NF.A.2.b (locate a/b on number line).
- Later: 4.NF.A.1 (the general rule via multiplying numerator and denominator by the same n).

**Recommended representations / models named in the Progressions**

- Number line diagrams showing fractions landing on the same point.
- Tape diagrams subdivided into smaller equal pieces.
- Visual fraction models more generally (the standard explicitly invokes them).
