# Progressions excerpt — 4.NF.A.1

**Source:** Progressions-3-5-Fractions.pdf, §Grade 4 — "Equivalent fractions," extracted pages 10–12.

## Verbatim excerpt

> Grade 4 students learn a fundamental property of equivalent fractions: multiplying the numerator and denominator of a fraction by the same non-zero whole number results in a fraction equal to the original fraction. This property forms the basis for much of the other work in Grade 4, including the comparison, addition, and subtraction of fractions and the introduction of finite decimals. (p. 11)

> Equivalent fractions. Students can use area representations, strips of paper, tape diagrams, and number line diagrams to reason about equivalence. 4.NF.1 They see that the numerical process of multiplying the numerator and denominator of a fraction by the same number, n, corresponds to partitioning each piece of the diagram into n smaller equal pieces (MP.1). Each region or length that represents a unit fraction (a subordinate unit) is partitioned into n smaller regions or lengths, each of which represents a unit fraction (a subordinate unit of a subordinate unit). The whole has then been partitioned into n times as many pieces, and there are n times as many smaller unit fraction pieces as in the original fraction. (pp. 11–12)

> This argument, once understood for a range of examples, can be seen as a general argument, working directly from the Grade 3 understanding of a fraction as a point on the number line. (p. 12)

> The fundamental property can be presented in terms of division, as in, e.g.
>
> 28/36 = (28 ÷ 4) / (36 ÷ 4) = 7/9.
>
> Because the equations 28 ÷ 4 = 7 and 36 ÷ 4 = 9 tell us that 28 = 4 × 7 and 36 = 4 × 9, this is the fundamental property in disguise:
>
> (4 × 7) / (4 × 9) = 7/9.
>
> It is possible to over-emphasize the importance of simplifying fractions in this way. There is no mathematical reason why fractions must be written in simplified form, although it may be convenient to do so in some cases. (p. 12)

The Progression illustrates with three diagrams (p. 11): an area representation showing 2/3 = (4×2)/(4×3); a tape diagram showing the same; and a number line showing 4/3 = (5×4)/(5×3).

Sidebar — the standard verbatim:

> 4.NF.1 Explain why a fraction a/b is equivalent to a fraction (n×a)/(n×b) by using visual fraction models, with attention to how the number and size of the parts differ even though the two fractions themselves are the same size. Use this principle to recognize and generate equivalent fractions. (p. 11)

## Extraction for drafters

**What the kid should be able to do**

- Explain why a/b = (n×a)/(n×b) using a visual fraction model — area, tape, or number line.
- Recognize that partitioning each existing piece into n smaller equal pieces yields n times as many pieces (and each is n times smaller).
- Generate equivalent fractions in both directions (e.g., 2/3 → 8/12, OR 28/36 → 7/9 via division).
- Attend to how the **number and size of parts differ** even though the fraction has the same size.

**Common misconceptions / developmental hurdles named in the Progressions**

- **Over-emphasizing simplification** — the Progressions explicitly warns: "There is no mathematical reason why fractions must be written in simplified form" (p. 12). Drafters should not treat "lowest terms" as a correctness requirement.
- **Treating the rule as a mnemonic ("multiply by 1") without justification** — the Progressions says using 7/9 = (7/9) × 1 = (7/9) × (4/4) = 28/36 "does not constitute a valid argument at this grade, since students have not yet learned fraction multiplication" (p. 12). The reasoning must come from the visual model of repartitioning, not from the algebraic identity.

**Where this skill sits in the progression**

- Earlier: 3.NF.A.3.a (equivalent = same point on number line / same size), 3.NF.A.3.b (recognize and generate simple equivalents via visual model).
- Later: 4.NF.A.2 (compare fractions with different numerators and denominators), 4.NF.B.3 (add/subtract fractions with like denominators using decomposition), 5.NF.A.1 (add/subtract with unlike denominators — relies on this rule to find common denominators).

**Recommended representations / models named in the Progressions**

- Area representations: partition each of the existing pieces into n smaller equal pieces; count the new total.
- Tape diagrams partitioned into 4×3 sub-pieces, etc.
- Number line diagrams: partition each interval of length 1/3 into 5 parts to see 4/3 = 20/15.
