# Progressions excerpt — 3.NF.A.3.a

**Source:** Progressions-3-5-Fractions.pdf, §Grade 3 — "Equivalent fractions," extracted pages 9–10.

## Verbatim excerpt

> Equivalent fractions. Grade 3 students do some preliminary reasoning about equivalent fractions, in preparation for work in Grade 4. As students experiment on number line diagrams they discover that many fractions label the same point on the number line, and are therefore equal; that is, they are equivalent fractions. For example, the fraction 1/2 is equal to 2/4 and also to 3/6. Students can also use tape diagrams to see fraction equivalence. 3.NF.3 (p. 9)

Sidebar — the standard verbatim:

> 3.NF.3 Explain equivalence of fractions in special cases, and compare fractions by reasoning about their size.
>
> a. Understand two fractions as equivalent (equal) if they are the same size, or the same point on a number line. (p. 9)

## Extraction for drafters

**What the kid should be able to do**

- Recognize that two fractions are equivalent if they label the same point on a number line, or represent the same size.
- Use a number line diagram or tape diagram to demonstrate that fractions like 1/2, 2/4, and 3/6 land on the same point.

**Common misconceptions / developmental hurdles**

- Not explicitly named in this section. The Progressions stresses (p. 10) that "comparisons are valid only when the two fractions refer to the same whole" — the same-whole hazard applies to equivalence too. Consult `fractions-misconceptions.json` for SM's project-internal list.

**Where this skill sits in the progression**

- Earlier: 3.NF.A.1 (a/b as a copies of 1/b), 3.NF.A.2 (fractions on the number line).
- Later: 3.NF.A.3.b (generate simple equivalent fractions), 3.NF.A.3.c (whole numbers as fractions), 4.NF.A.1 (general rule a/b = (n×a)/(n×b)).

**Recommended representations / models named in the Progressions**

- Number line diagrams — same point ⇒ equivalent fractions.
- Tape diagrams — same shaded length ⇒ equivalent fractions.
- Area representations (implicit from the Grade 3 section on 3.NF.1).
