# Progressions excerpt — 4.NF.A.2

**Source:** Progressions-3-5-Fractions.pdf, §Grade 4 — "Equivalent fractions" (comparison subsection), extracted page 12.

## Verbatim excerpt

> Grade 4 students use their understanding of equivalent fractions to compare fractions with different numerators and different denominators. 4.NF.2 For example, to compare 5/8 and 7/12 they rewrite both fractions:
>
> (12 × 5) / (12 × 8) = 60/96 and (7 × 8) / (12 × 8) = 56/96.
>
> Because 60/96 and 56/96 have the same denominator, students can compare them using Grade 3 methods and see that 56/96 is smaller, so
>
> 7/12 < 5/8. (p. 12)

> Students also reason using benchmarks such as 1/2 and 1. For example, they see that 7/8 < 13/12 because 7/8 is less than 1 (and is therefore to the left of 1) but 13/12 is greater than 1 (and is therefore to the right of 1). (p. 12)

> Grade 4 students who have learned about fraction multiplication can see equivalence as "multiplying by 1":
>
> 7/9 = (7/9) × 1 = (7/9) × (4/4) = 28/36
>
> However, although a useful mnemonic device, this does not constitute a valid argument at this grade, since students have not yet learned fraction multiplication. (p. 12)

Sidebar — the standard verbatim:

> 4.NF.2 Compare two fractions with different numerators and different denominators, e.g., by creating common denominators or numerators, or by comparing to a benchmark fraction such as 1/2. Recognize that comparisons are valid only when the two fractions refer to the same whole. Record the results of comparisons with symbols >, =, or <, and justify the conclusions, e.g., by using a visual fraction model. (p. 12)

## Extraction for drafters

**What the kid should be able to do**

- Compare two fractions with different numerators AND different denominators.
- Use at least one of three strategies named in the standard:
  1. Create a common denominator (rewrite both fractions with the same denominator).
  2. Create a common numerator.
  3. Compare to a benchmark fraction such as 1/2 or 1.
- Record comparisons with >, =, <.
- Justify the conclusion with a visual fraction model.
- Recognize that comparisons are only valid when both fractions refer to the same whole.

**Common misconceptions / developmental hurdles named in the Progressions**

- **Same-whole hazard** — the standard explicitly repeats the 3.NF.A.3.d rule: comparisons are valid only when the two fractions refer to the same whole.
- **Premature use of "multiply by 1" as justification** — the Progressions flags this as "a useful mnemonic device" but explicitly says "this does not constitute a valid argument at this grade, since students have not yet learned fraction multiplication" (p. 12). Drafters should ground reasoning in repartitioning of visual models, not in algebraic identity.

**Where this skill sits in the progression**

- Earlier: 3.NF.A.3.d (compare fractions with same numerator OR same denominator), 4.NF.A.1 (the general equivalence rule a/b = (n×a)/(n×b)).
- Later: 4.NF.B.3 (add/subtract with like denominators — needs equivalence to handle decomposition), 5.NF.A.1 (add/subtract fractions with unlike denominators — depends on creating common denominators).

**Recommended representations / models named in the Progressions**

- Visual fraction models (named in the standard).
- Number line diagrams — benchmark reasoning ("7/8 is to the left of 1, 13/12 is to the right").
- Common-denominator rewrites using the 4.NF.A.1 equivalence rule.
- Common-numerator rewrites as an alternative strategy.
