# Progressions excerpt — 3.NF.A.2.a

**Source:** Progressions-3-5-Fractions.pdf, §Grade 3 — "The number line and number line diagrams," extracted pages 8–9.

## Verbatim excerpt

> On a number line diagram, the whole is the unit interval, that is, the interval from 0 to 1, measured by length. Iterating this whole to the right marks off the whole numbers, so that the intervals between consecutive whole numbers, from 0 to 1, 1 to 2, 2 to 3, etc., are all of the same length, as shown. Students might think of the number line as an infinite ruler with the unit interval as the unit of measurement. (p. 9)

> To construct a unit fraction on a number line diagram, e.g., 1/3, students partition the unit interval into 3 intervals of equal length and recognize that each has length 1/3. They determine the location of the number 1/3 by marking off this length from 0 [...]. 3.NF.2 (p. 9)

> Although number line diagrams are important representations for students as they develop an understanding of a fraction as a number, initially they use other representations such as area representations, strips of paper, and tape diagrams. These, like number line diagrams, can be subdivided, representing an important aspect of fractions. (p. 9)

Sidebar — the standard verbatim:

> 3.NF.2 Understand a fraction as a number on the number line; represent fractions on a number line diagram.
>
> a. Represent a fraction 1/b on a number line diagram by defining the interval from 0 to 1 as the whole and partitioning it into b equal parts. Recognize that each part has size 1/b and that the endpoint of the part based at 0 locates the number 1/b on the number line. (p. 9)

## Extraction for drafters

**What the kid should be able to do**

- Define the interval from 0 to 1 on a number line as "the whole."
- Partition the unit interval into b equal parts and identify each as length 1/b.
- Locate the point 1/b on the number line as the right-hand endpoint of the first part (the part based at 0).

**Common misconceptions / developmental hurdles**

- Not explicitly named in this section. The Progressions emphasizes that area, strip, and tape representations should come first — jumping to number lines too fast is a hazard (p. 9). Consult `fractions-misconceptions.json` for SM's project-internal list (likely includes treating tick-marks as labels rather than as endpoints of equal-length intervals).

**Where this skill sits in the progression**

- Earlier: 3.NF.A.1 (a fraction 1/b as one of b equal parts of a whole), 3.MD.4 (rulers marked in halves and fourths — direct measurement bridge).
- Later: 3.NF.A.2.b (locate a/b on the number line), 3.NF.A.3 (equivalence and comparison using the number line).

**Recommended representations / models named in the Progressions**

- Number line diagrams with the unit interval explicitly defined.
- "Infinite ruler" metaphor — the number line is built from iterating the unit interval.
- Tape diagrams and area representations as bridges to the number line.
